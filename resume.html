Featured Project

Legal Aid AI — Multilingual RAG Legal Assistant

Full Stack · AI/ML · Social Impact 2024 — Present

End-to-end RAG system that helps low-income Indian citizens understand their legal rights in English, Hindi, and Marathi. Ships with a built-in knowledge base of 14 major Indian laws (Constitution, BNS, BNSS, RTI, Consumer Protection, Domestic Violence Act, and more) — works immediately after deployment with zero manual data entry.

  • Built a production RAG pipeline: PDF ingestion → PyMuPDF parsing → RecursiveCharacterTextSplitter chunking → SentenceTransformer embeddings → FAISS vector search → Groq LLaMA-3 generation with structured citations (Act · Section · Page).
  • Designed and shipped a 14-law built-in knowledge base (99 sections) covering Constitution of India, BNS 2023, BNSS 2023, BSA 2023, RTI Act, Consumer Protection Act, Domestic Violence Act, Labour Laws, Motor Vehicles Act, IT Act, Environmental Law, Property Law, Family Law, and Tenant Rights — available on day one with no uploads.
  • Implemented multilingual domain classification across 16 legal domains using keyword-scoring with 150+ English, Hindi, and Marathi triggers — 8/8 test cases correct without any ML model dependency.
  • Architected a live PDF ingestion admin API (`POST /admin/upload-law`) that parses, chunks, and merges new laws into the FAISS index at runtime — no server restart required.
  • Built a React 18 frontend with dark mode, multilingual UI (EN/HI/MR), auto-growing textarea, voice input via Groq Whisper, expandable source citations, NALSA center locator by GPS, and typing animation.
  • Designed the system to run entirely free: local `all-MiniLM-L6-v2` embeddings (no API key needed), Groq LLaMA-3 for generation (free tier), with graceful rule-based fallback when no key is configured.
  • Deployed with Docker + Nginx reverse proxy; Render-ready via `render.yaml`; rate-limited with in-memory middleware; CORS-configured for production origins.
